app.use() is used to register middleware in an Express application.

👉 Middleware = functions that run between request and response.

Think of a request like this:

Client → [ middleware ] → [ route handler ] → Response


app.use() decides what runs in the middle.
A request handler is any function that processes an incoming request in Express, including middleware and route handlers. A route handler is a specific type of request handler that is bound to a particular HTTP method and path and usually sends the final response.
